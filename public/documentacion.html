Documentación de Endpoints

Registro de Usuario

Descripción: Endpoint para registrar un nuevo usuario.

Método: POST

Ruta: /registro

Parámetros:

  • username (cadena, requerido): Nombre de usuario
  • password (cadena, requerido): Contraseña

Iniciar Sesión

Descripción: Endpoint para iniciar sesión.

Método: POST

Ruta: /login

Parámetros:

  • username (cadena, requerido): Nombre de usuario
  • password (cadena, requerido): Contraseña
